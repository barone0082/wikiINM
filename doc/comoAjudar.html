Inmetrics Code-Wiki

Como enviar seu código para o code-Wiki:

1 - Baixe abaixo o padrão de classes ruby/java de envio.

                        
2 - Clone o projeto do git em algum diretório em seu desktop.

3 - Dentro do diretório codewiki\code crie uma pasta com o nome da funcionalidade de seu código, dentro dela deverá conter sua classe criada.

OBS: Para que a documentação seja gerada de maneira correta, favor manter o nome da classe com o padrão de nomenclatura:

xxxxx_java.java   ou   xxxxx_ruby.rb

4 - Tudo feito? codigo e comentarios criados? Agora basta apenas atualizar o repositorio no Git.
